3.js实现web中的VR应用，需要什么
  • 设备信息
  • 图像的变换

    • WebVR

    什么是WebVR

    • 一套实验性的JS VR api
    • 目前处于草案阶段
    • ThreeJS本身很难拿到设备信息，因为WebGL就没这个能力，但是WebVR是可以的

    WebVR可以做什么

    • 对VR来说很重要
    • 提供了一套获取HMD设备信息的实现要求
    • 换句话说实现了它的浏览器就可以直接通过这个API获取设备信息

    WebVR Compatibility

    WebVR Polyfill by GoogleVR

    webvr的支持程度目前来看还很差: currently available in

    • Firefox Nightly builds
    • Chrome 56+ for Android
    • experimental builds of Chromium for Windows
    • Samsung Internet Browser for Gear VR

    可以通过googleVR 的 polyfill来解决

    AR

    AR(增强现实)的概念/原理介绍 AR是计算机结合自然环境生成内容，并结合显示在自然环境中的一种技术

    几个要素

    1. 获取视频流
    2. 分析视频中的 标记（marker）
    3. 在对应标记处渲染放置3D图像
    4. 叠加3D图像和视频

    JSARToolKit

    可以帮助我们处理视频流中的标记

    1. 实时计算用户的视窗范围
    2. 找到视窗中的marker（并且可以处理自然标记）

    Three.js

    这几个任务里面，3d图形是threejs的强项

    自然，需要用threejs处理这个部分

    Step BY Step

    大概有四个步骤

    1. 通过WebCam获取视频
    2. 通过ARToolKit 找到视窗中的marker（并且可以处理自然标记）
    3. 通过ThreeJS，生成3D图形
    4. 叠加在现实世界和3d
